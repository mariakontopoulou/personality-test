Crafted by Maria Kontopoulou

The Personality Test project was developed as part of the Coding course at Audencia Business School.
The main objective of this project is to integrate Typeform quizzes into a custom website built with HTML and CSS.
The website features four personality tests, each containing five questions, designed using Typeform’s “Outcome Quiz” method to generate personalized results.
